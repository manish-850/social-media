# Social Media Web App

A server-rendered social media application built with Node.js, Express,
EJS, and JavaScript. This project provides core social features and is
structured for scalability and future database integration.

## Live Demo

https://social-media-n91w.onrender.com

## Features

-   Server-side rendering using EJS
-   Express.js backend
-   Structured MVC-like folder organization
-   Static asset management via public directory
-   Ready for authentication and database integration

## Tech Stack

-   Node.js
-   Express.js
-   EJS
-   JavaScript
-   CSS

## Installation

1.  Clone the repository:

        git clone https://github.com/manish-850/social-media.git

2.  Navigate into the project folder:

        cd social-media

3.  Install dependencies:

        npm install

4.  Create a .env file:

       1. MONGO_URI
       2. JWT_SECRET_KEY

5.  Start the server:

        npm start

6.  Open in browser:

        http://localhost:3000
